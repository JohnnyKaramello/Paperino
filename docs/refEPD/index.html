Micro EPD (ePaper display 1.1”)

EPaper displays (EPDs) mimic the appearance of natural paper. They can hold text and images indefinitely, even without electricity. They are thus perfect for ultra-low power applications which do not require too high update frequency.

Paperino uses an EPD from Plastic Logic. This is an Eink-based active-matrix display based on the Ultrachip UC8156 EPD controller. The display supports 148 x 70 pixels.

Specification

Type Value
Resolution 148 x 70 px
Active area 25.16 x 11.9 mm
Pixel density 150 ppi
Grey levels 4
Weight 1.2g
Thickness 460 µm
Operating conditions 0C to 40C
Storage conditions -25C to 50C
